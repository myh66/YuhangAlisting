# Code of Conduct

## Our Pledge

We want YuhangAlisting to be a practical and welcoming project for users and
contributors. Everyone is expected to communicate with patience, respect, and a
focus on improving the software.

## Expected Behavior

- Be respectful when discussing bugs, design choices, and tradeoffs.
- Assume good intent and ask for clarification before escalating tone.
- Keep issues and pull requests focused on concrete project work.
- Give credit when building on someone else's ideas or code.

## Unacceptable Behavior

- Harassment, insults, discrimination, or personal attacks.
- Publishing private information without permission.
- Repeated off-topic comments that make project maintenance harder.
- Any conduct that would reasonably make others feel unsafe or unwelcome.

## Enforcement

Maintainers may edit, hide, or remove comments and may temporarily or
permanently restrict participation for behavior that violates this code.

For security-sensitive reports, use the process in [SECURITY.md](SECURITY.md)
instead of public issues.
